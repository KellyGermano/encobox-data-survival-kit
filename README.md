# Encobox Data Survival Kit

Este proyecto es parte de la implementación del primer ecosistema de datos en **Encobox**, una empresa de logística urbana.  
El objetivo es demostrar el ciclo completo de un analista de datos:

1. **Extracción y Limpieza (ETL básico):**  
   Procesamiento de datos de pedidos, clientes y repartidores con Python (Pandas).

2. **Análisis Exploratorio (EDA):**  
   - Clientes más importantes (análisis RFM).  
   - Zonas con mayor volumen de pedidos (mapas de calor).  
   - Identificación de horas y días pico.  

3. **Visualización (Dashboard BI):**  
   Dashboard interactivo en **Power BI** / **Streamlit** con insights clave para la planificación logística.

4. **Automatización de Reportes:**  
   Script en Python para generar informes de ventas semanales y ahorrar horas de trabajo manual.

5. **Predictivo (PoC):**  
   Modelo de predicción de la demanda de pedidos usando series temporales.

---

### Tecnologías principales
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Power BI / Streamlit  
- GitHub para control de versiones
